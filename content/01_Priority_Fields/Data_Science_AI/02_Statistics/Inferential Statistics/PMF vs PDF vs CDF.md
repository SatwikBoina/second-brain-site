#  Comparison

| Ground                       | [[Probability Mass Function]] | [[Probability Density Function]] | [[Cumulative Distribution Function]] |
| ---------------------------- | ----------------------------- | -------------------------------- | ------------------------------------ |
| Type of variable             | Discrete                      | Continuous                       | Both                                 |
| What it represents           | Probability at exact values   | Density of probability           | Cumulative probability               |
| Probability at a point       | Non-zero                      | Always zero                      | Defined                              |
| How probability is obtained  | Directly from function        | Area under curve                 | Function value                       |
| Mathematical meaning         | Mass                          | Density                          | Accumulation                         |
| Output range                 | 0 to 1                        | ≥ 0                              | 0 to 1                               |
| Total probability            | Sum of all values = 1         | Total area = 1                   | Max value = 1                        |
| Monotonic behavior           | No                            | No                               | Always non-decreasing                |
| Used to compute              | P(X = x)                      | P(a ≤ X ≤ b)                     | P(X ≤ x)                             |
| Works with intervals         | No                            | Yes                              | Yes                                  |
| Works with exact values      | Yes                           | No                               | Yes                                  |
| Shape interpretation         | Bars / spikes                 | Smooth curve                     | Increasing curve                     |
| Common examples              | Dice, counts                  | Height, time                     | Percentiles, p-values                |
| Role in inference            | Rare                          | Moderate                         | Dominant                             |
| Used in hypothesis testing   | No                            | Indirect                         | Yes                                  |
| Used in confidence intervals | No                            | Indirect                         | Yes                                  |
| Used in p-values             | No                            | No                               | Yes                                  |
| Primary intuition            | “How much probability at x?”  | “How dense is uncertainty?”      | “How much probability up to x?”      |
