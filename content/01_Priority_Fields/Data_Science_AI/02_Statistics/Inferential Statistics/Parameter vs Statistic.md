The fundamental difference lies in the scope of the data those terms describe.

## Definition:
- **Parameter :** 
	- Population characteristic.
	- An unknown numerical fact about an entire population. It is the ultimate truth.
	- It is always a fixed constant.
- **Statistic :** 
	- Sample characteristic.
	- It is the variable estimate of the fact derived from a sample.
	- It is random variable because it changes w.r.t to the sample chosen.


|**Feature**|**Population Parameter (The Truth)**|**Sample Statistic (The Estimate)**|
|---|---|---|
|**Scope**|Entire Population|Subset (Sample)|
|**Nature**|**Constant**, but usually unknown|**Random Variable**, changes with every sample|
|**Notation Convention**|**Greek** Letters|**Roman** Letters (or hats ^)|
|**Mean (Average)**|**$\mu$** (Mu)|**$\bar{x}$** (x-bar)|
|**Standard Deviation**|**$\sigma$** (Sigma)|**$s$**|
|**Proportion/Probability**|**$p$** or $\pi$|**$\hat{p}$** (p-hat)|
